# Creative Agency Templates

## Creative Brief Template

```markdown
# Creative Brief: [Project Name]

## Client
Client name and background.

## Project Overview
Brief description of the project.

## Objectives
- Objective 1
- Objective 2

## Target Audience
Who is this for?

## Key Messages
- Message 1
- Message 2

## Deliverables
- [ ] Deliverable 1
- [ ] Deliverable 2

## Timeline
- Milestone 1: Date
- Milestone 2: Date

## Budget
Budget constraints.

## Brand Guidelines
Link to brand guidelines or key requirements.

## References
- Reference 1
- Reference 2

## Approval
- [ ] Client approved
- [ ] Creative Director approved
```

## Art Direction Template

```markdown
# Art Direction: [Project Name]

## Visual Direction
Description of visual approach.

## Color Palette
- Primary: #HEX
- Secondary: #HEX
- Accent: #HEX

## Typography
- Headlines: Font Name
- Body: Font Name

## Imagery Style
Description of image style.

## Motion Style
Description of motion/animation approach.

## Audio Direction
Description of audio approach.

## Dos and Don'ts
- ✅ Do this
- ❌ Don't do this

## References
Visual references and examples.
```

## Asset Specification Template

```markdown
# Asset Spec: [Asset Name]

## Type
Video | Image | Audio | Animation

## Dimensions
Width x Height (pixels)

## Duration
Duration (for video/audio)

## Format
- Delivery format: MP4, PNG, etc.
- Source format: Project files

## Technical Requirements
- Frame rate: 30fps
- Resolution: 1080p
- Audio: Stereo, 48kHz

## Usage
Where will this be used?

## Variations
- Variation 1: Description
- Variation 2: Description

## Accessibility
- [ ] Alt text provided
- [ ] Captions included
- [ ] Audio description available
```

## Delivery Checklist Template

```markdown
# Delivery Checklist: [Project Name]

## Assets
- [ ] Asset 1 - Approved
- [ ] Asset 2 - Approved
- [ ] Asset 3 - Approved

## Formats
- [ ] Primary format delivered
- [ ] Web-optimized format delivered
- [ ] Source files archived

## Documentation
- [ ] Asset manifest provided
- [ ] Usage guidelines provided

## Accessibility
- [ ] Alt text for all images
- [ ] Captions for all videos
- [ ] Transcripts for audio

## Quality Check
- [ ] Brand guidelines verified
- [ ] Technical specs verified
- [ ] Client approval received

## Delivery
- [ ] Files uploaded to delivery platform
- [ ] Client notified
- [ ] Project archived
```
