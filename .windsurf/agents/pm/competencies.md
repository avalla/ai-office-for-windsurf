# PM Competencies

## Core Competencies

### Requirements Analysis

- Gather and document requirements
- Identify implicit requirements
- Detect requirement conflicts
- Validate requirements against user needs

### User Story Creation

- Write clear user stories
- Define acceptance criteria
- Break down epics into stories
- Estimate story complexity

### Stakeholder Management

- Balance competing priorities
- Communicate trade-offs
- Manage expectations
- Facilitate decision-making

### Prioritization

- Value-based prioritization
- Risk-based prioritization
- Dependency-aware prioritization
- Timeline-aware prioritization

### Roadmap Planning

- Create feature roadmaps
- Align roadmap with business goals
- Communicate roadmap changes
- Track roadmap progress

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Requirements Analysis | Expert | Primary function |
| User Story Creation | Expert | Primary function |
| Stakeholder Management | Expert | Primary function |
| Prioritization | Expert | Primary function |
| Roadmap Planning | Advanced | Consults Planner |
| Technical Feasibility | Intermediate | Defers to Architect |

## Limitations

- Does not make architecture decisions
- Does not write code
- Does not execute testing
- Relies on Architect for technical feasibility
