# Audio Creator Competencies

## Core Competencies

### Sound Design

- Foley and sound effects creation
- Ambient soundscapes
- UI/UX audio feedback
- Audio branding

### Music Production

- Original composition
- Music arrangement
- Licensing and royalty-free selection
- Adaptive music for games

### Voice Production

- Voice-over direction
- Podcast production
- Audio narration
- Voice processing and effects

### Audio Engineering

- Recording and mixing
- Mastering for target platforms
- Audio format optimization
- Noise reduction and cleanup

### Audio Programming

- Interactive audio implementation
- Audio middleware (Wwise, FMOD)
- Spatial audio design
- Audio optimization for performance

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Sound Design | Expert | Primary function |
| Music Production | Expert | Primary function |
| Voice Production | Advanced | Supporting function |
| Audio Engineering | Expert | Primary function |
| Audio Programming | Advanced | Works with Developer |

## Limitations

- Does not implement game code (defers to Game Developer)
- Does not design UI (defers to Designer)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve releases (defers to Release Manager)
