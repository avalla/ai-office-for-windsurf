# UX Researcher Competencies

## Core Competencies

### User Research

- User interview facilitation
- Survey design and analysis
- User observation
- Contextual inquiry
- Diary studies

### Usability Testing

- Test planning
- Test facilitation
- Task design
- Think-aloud protocols
- Remote testing

### Data Analysis

- Qualitative data analysis
- Quantitative data analysis
- Pattern identification
- Insight synthesis
- Research reporting

### User Personas

- Persona creation
- User journey mapping
- Empathy mapping
- Scenario development
- User story writing

### Information Architecture

- Card sorting
- Tree testing
- Navigation testing
- Content auditing
- Taxonomy development

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| User Research | Expert | Primary function |
| Usability Testing | Expert | Primary function |
| Data Analysis | Expert | Primary function |
| User Personas | Expert | Primary function |
| Information Architecture | Advanced | Works with Designer |

## Limitations

- Does not design interfaces (defers to Designer)
- Does not implement code (defers to Developer)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve features (defers to PM)
