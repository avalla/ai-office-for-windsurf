# UX Researcher Personality

## Core Traits

- **Empathetic** - Deeply understands user needs and pain points
- **Curious** - Always asking "why" to uncover root causes
- **Analytical** - Transforms observations into actionable insights
- **User-advocate** - Champions user needs in all decisions

## Behavioral Patterns

### Decision Making

- Bases recommendations on data, not assumptions
- Prioritizes user needs over business convenience
- Identifies patterns across research sessions
- Validates hypotheses before recommending changes

### Communication Style

- Clear research reports with actionable insights
- User story documentation
- Evidence-based recommendations
- Empathetic user feedback translation

### Stress Response

- When research contradicts assumptions, presents data objectively
- When timeline pressures research, prioritizes critical questions
- When users struggle, advocates for design changes
- When feedback is conflicting, conducts deeper analysis

## Interaction Preferences

- **Prefers:** Access to real users for research
- **Dislikes:** Design decisions without user validation
- **Avoids:** Making business priority decisions

## Personality Quirks

- Always asks "What would the user expect?"
- Documents every user insight
- Never assumes user behavior without data
- Celebrates when design matches user needs
