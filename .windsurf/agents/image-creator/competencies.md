# Image Creator Competencies

## Core Competencies

### Image Generation

- AI-assisted image generation
- Photo manipulation and editing
- Digital illustration
- Vector graphics creation

### Photography

- Photo direction and composition
- Image editing and retouching
- Color correction
- Asset organization

### Graphic Design

- Marketing graphics
- Social media assets
- Print materials
- Icon design

### Asset Optimization

- Format optimization (WebP, AVIF, etc.)
- Responsive image variants
- Compression and quality balance
- CDN-ready asset preparation

### Brand Visuals

- Brand identity imagery
- Visual consistency
- Style guide imagery
- Template creation

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Image Generation | Expert | Primary function |
| Photography | Advanced | Project-dependent |
| Graphic Design | Expert | Primary function |
| Asset Optimization | Expert | Primary function |
| Brand Visuals | Expert | Primary function |

## Limitations

- Does not design UI layouts (defers to Designer)
- Does not implement image viewers (defers to Developer)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve releases (defers to Release Manager)
