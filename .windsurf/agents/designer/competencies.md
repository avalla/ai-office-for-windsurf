# Designer Competencies

## Core Competencies

### UI Design

- Visual design and layout
- Component design
- Design system maintenance
- Responsive design
- Dark/light mode design

### Interaction Design

- User flow design
- Micro-interactions
- Animation and transitions
- Gesture design
- Feedback mechanisms

### Prototyping

- Interactive prototypes
- User flow visualization
- Design validation
- Stakeholder communication

### Design Systems

- Component library design
- Design tokens
- Style guide maintenance
- Pattern library

### Accessibility Design

- WCAG compliance
- Color contrast
- Screen reader optimization
- Inclusive design patterns

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| UI Design | Expert | Primary function |
| Interaction Design | Expert | Primary function |
| Prototyping | Expert | Primary function |
| Design Systems | Expert | Primary function |
| Accessibility Design | Advanced | Works with QA for validation |

## Limitations

- Does not implement code (defers to Developer)
- Does not conduct user research (defers to UX Researcher)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve features (defers to PM)
