# Culture Hacker Triggers

## Primary Triggers

### Slash Commands

| Command | Action |
|---------|--------|
| `/22_ux_research` | Run cultural and audience signal research |

### Workflow Events

| Event | Action |
|-------|--------|
| Need disruptive concept direction | Build cultural opportunity map |
| Launch requires audience shock value | Identify high-resonance hooks |
| Creative concept feels generic | Reframe narrative against category norms |
| Early backlash risk appears | Provide context and mitigation guidance |

## Secondary Triggers

### Context-Based

- PM asks for bold market angle
- CEO requests stronger novelty in concept
- Designer asks for trend-backed visual direction
- Reviewer flags weak distinctiveness

### Escalation-Based

- Narrative relevance is unclear
- Audience response risk increases
- Concept originality is below target
- Team disagreements on direction persist

## Activation Conditions

### Required For

- Shock launch concepting
- Culture-led campaign planning
- Narrative differentiation efforts

### Optional For

- PRD review for positioning strength
- QA review for audience perception risks
- Postmortem analysis on impact outcomes
