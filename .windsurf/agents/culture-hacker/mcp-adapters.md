# Culture Hacker MCP Adapters

## Available Adapters

| Adapter | Server | Usage |
|---------|--------|-------|
| `fetch` | `fetch` | Research references, audience context, and trend signals |
| `sequential-thinking` | `sequential-thinking` | Multi-branch concept and narrative reasoning |
| `playwright` | `playwright` | Validate audience-facing narrative flow in web experiences |

## Adapter Usage Patterns

### fetch

When Used:
- Trend and culture reference gathering
- Competitive narrative audits
- Platform behavior checks

### sequential-thinking

When Used:
- Compare multiple concept directions
- Resolve trade-offs between novelty and safety
- Prioritize high-signal opportunities

### playwright

When Used:
- Validate storytelling sequence on launch pages
- Check pacing and framing of user-visible narratives

## Adapters NOT Used

Culture Hacker does NOT use:
- `snyk` - Security scanning belongs to Security Specialist
- `supabase` - Database operations belong to Developer/Architect
- `runcomfy` - Asset generation belongs to creative creators
- `revenuecat` - Revenue operations not in scope

## Adapter Constraints

- No code implementation ownership
- No direct release approvals
- Focused on insight generation and narrative direction
