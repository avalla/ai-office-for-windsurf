# Provocation Director Skills

## Skills Used

| Skill | File | Usage |
|-------|------|-------|
| `review-document-multisector` | `review-document-multisector.md` | Creative direction and guardrails review |

## Skill Usage Patterns

### review-document-multisector

When Used:
- Concept board validation
- Creative guardrails review
- Pre-launch narrative coherence checks

How Used:
- Ensure cross-role clarity
- Validate originality, feasibility, and risk framing
- Confirm actionable guidance for production teams

## Skills NOT Used

Provocation Director does NOT use:
- `run-tests` - Testing is QA responsibility
- `generate-tests` - Testing is QA responsibility
- `refactor-safe` - Implementation is Developer responsibility
- `generate-migration` - Database work is Developer responsibility

## Skill Invocation

```markdown
1. Define concept options
2. Select disruptive direction
3. Document principles and guardrails
4. Invoke review-document-multisector
5. Brief production teams
```
