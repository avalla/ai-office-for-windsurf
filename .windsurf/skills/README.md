# Skills

Skills are reusable micro-playbooks used by workflows.

Core skills:
- analyze-repo
- trace-request-to-code
- run-tests-and-summarize
- generate-tests
- review-diff
- security-quickcheck
- postmortem-update
- task-management
- project-analysis

Task Management:
- Handles task state transitions between TODO/WIP/REVIEW/DONE/ARCHIVED
- Validates assignments and prevents assignment loops
- Integrates with workflows for automated task movement
- Tracks assignment history and team compatibility

Project Analysis:
- Analyzes existing project structure and technology stack
- Detects team patterns and expertise areas
- Imports existing tasks and documentation
- Generates framework-compatible artifacts
- Supports import workflow decision making
