# Memory

Project memory, patterns, and learned insights.

## Purpose

This folder stores:
- **Patterns** - Reusable solutions and design patterns
- **Insights** - Key learnings and discoveries
- **Decisions** - Important project decisions and rationale
- **Knowledge** - Domain-specific knowledge and expertise

## Structure

- `patterns/` - Design patterns, code patterns, architectural patterns
- `insights/` - Key learnings and discoveries
- `decisions/` - Important decisions and their rationale
- `knowledge/` - Domain-specific information and expertise

## Usage

1. **Document patterns** as they emerge
2. **Capture insights** during development and retrospectives
3. **Record decisions** with their context and reasoning
4. **Build knowledge** base for the project domain

## Benefits

- Faster onboarding for new team members
- Consistent application of learned patterns
- Preservation of institutional knowledge
- Better decision-making with historical context

## Maintenance

Review and update memory items regularly to ensure they remain relevant and accurate.
