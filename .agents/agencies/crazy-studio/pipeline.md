## Fast Viral Concept Pipeline

For a short-cycle disruptive concept:

```
Router → PM Brief → Culture Hacker Scan → Provocation Direction → Asset Production → QA → Launch
```

## Parallel Workstreams

| Phase | Parallel Workflows |
|-------|-------------------|
| Creative Development | Culture Hacker research + Provocation concepting |
| Production | Video + Audio + Image creation |
| Final Validation | QA quality + Security risk checks |

## Checkpoints

| Checkpoint | Trigger | Required Artifacts |
|------------|---------|-------------------|
| Brief Locked | PM and CEO align | Shock brief |
| Concept Locked | Provocation Director signs off | Concept board |
| Production Ready | Designer approves system | Storyboard + style guide |
| Risk Cleared | QA + Security pass | Risk and QC report |
| Launch Approved | CEO + Release Manager pass | Launch package |
