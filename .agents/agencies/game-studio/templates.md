# Game Studio Templates

## Game Design Document (GDD) Template

```markdown
# GDD: [Game Name]

## Overview
Brief description of the game.

## Vision
What is the core experience?

## Target Platform
- Primary: PC/Console/Mobile
- Secondary: Other platforms

## Genre
Game genre and sub-genre.

## Target Audience
Who is this game for?

## Core Mechanics

### Mechanic 1: [Name]
Description of core mechanic.

### Mechanic 2: [Name]
Description of core mechanic.

## Game Loop
Core gameplay loop description.

## Progression
How does the game progress?

## Narrative
Story and world description.

## Art Style
Visual direction and style.

## Audio Direction
Sound and music direction.

## Technical Requirements
- Engine: Unity/Unreal/Godot
- Performance targets
- Platform requirements

## Monetization
- Free-to-play / Premium
- In-app purchases
- DLC plans

## Milestones
- Prototype: Date
- Alpha: Date
- Beta: Date
- Release: Date
```

```markdown
- CI/CD setup
```

## Playtest Report Template

```markdown
# Playtest Report: [Build Version]

## Test Session
- Date: YYYY-MM-DD
- Duration: X hours
- Testers: X players

## Build Info
- Version: X.X.X
- Platform: PC/Console/Mobile
- Build Type: Alpha/Beta/RC

## Test Focus
What was being tested.

## Findings

### Critical Issues
| ID | Issue | Severity | Status |
|----|-------|----------|--------|
| C1 | Description | Critical | Open |

### Major Issues
| ID | Issue | Severity | Status |
|----|-------|----------|--------|
| M1 | Description | Major | Open |

### Minor Issues
| ID | Issue | Severity | Status |
|----|-------|----------|--------|
| m1 | Description | Minor | Open |

## Positive Feedback
What worked well.

## Player Feedback
Direct quotes and observations.

## Recommendations
Suggested improvements.

## Next Steps
Action items from this playtest.
```
