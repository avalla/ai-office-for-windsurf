## Campaign Pipeline (Multi-Asset)

For campaigns with multiple deliverables:

```
Brief → Art Direction → [Video Team + Audio Team + Image Team] → Integration → QA → Delivery
```

**Asset Types:**
- Hero video
- Social media cuts
- Static graphics
- Audio assets
- Interactive elements

## Fast-Track Pipeline (Single Asset)

For single asset requests:

```
Router → PM (Brief) → Creator (Produce) → QA (Review) → Delivery
```

## Revision Pipeline

```
Client Feedback → Designer (Art Direction Update) → Creator (Revise) → QA (Review) → Delivery
```

## Parallel Production

Creative work often runs in parallel:

| Phase | Parallel Workflows |
|-------|-------------------|
| Production | Video + Audio + Image creators |
| Review | QA (assets) + Designer (brand) |

## Checkpoints

| Checkpoint | Trigger | Required Artifacts |
|------------|---------|-------------------|
| Brief Approved | CEO approves | Creative brief |
| Art Direction Set | Designer completes | Style guide, references |
| Assets Ready | Creators complete | All asset files |
| QA Complete | QA approves | Review report |
| Delivery Ready | All approved | Final deliverables |
