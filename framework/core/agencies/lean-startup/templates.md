# Lean Startup Templates

## Hypothesis Template

```markdown
# Hypothesis: [Feature/Experiment Name]

## Problem
What problem are we solving?

## Hypothesis
We believe that [building this feature] for [these users]
will result in [this outcome].

## Success Metrics
- Primary metric: Target value
- Secondary metric: Target value

## Minimum Viable Feature
What is the smallest thing we can build to test this?

## Timeline
- Build: X days
- Measure: X days
- Learn: X days

## Risk
What if we're wrong?

## Decision Criteria
- Persevere if: [conditions]
- Pivot if: [conditions]
- Kill if: [conditions]
```

## MVP Feature Template

```markdown
# MVP Feature: [Name]

## User Story
As a [user], I want [action] so that [benefit].

## Acceptance Criteria (Minimal)
- [ ] Core functionality works
- [ ] Happy path complete

## Out of Scope
- Edge cases
- Advanced features
- Polish

## Implementation Notes
Quick implementation approach.

## Time Box
Maximum time to build: X hours/days.

## Success Indicator
How will we know this works?
```

## Metrics Dashboard Template

```markdown
# Metrics: [Project Name]

## North Star Metric
Primary success metric: [value]

## Key Metrics

| Metric | Current | Target | Status |
|--------|---------|--------|--------|
| Signups | X | Y | 🔄 |
| Activation | X% | Y% | 🔄 |
| Retention | X% | Y% | 🔄 |
| Revenue | $X | $Y | 🔄 |

## Weekly Trends
- Week 1: Data
- Week 2: Data
- Week 3: Data

## Insights
What we learned from the data.

## Actions
What we're doing based on insights.
```

## Pivot/Persevere Template

```markdown
# Decision: [Feature/Project Name]

## Date
YYYY-MM-DD

## Hypothesis
Original hypothesis statement.

## Results
What did we measure?

## Learning
What did we learn?

## Decision
- [ ] **Persevere:** Continue with current direction
- [ ] **Pivot:** Change direction (describe new direction)
- [ ] **Kill:** Stop this feature/project

## Rationale
Why this decision?

## Next Steps
What happens next?

## New Hypothesis (if pivoting)
New hypothesis statement.
```

## Sprint Retrospective Template

```markdown
# Sprint Retro: [Sprint Number]

## Duration
Start - End dates

## Goals
What we aimed to achieve.

## What We Shipped
- Feature 1
- Feature 2

## Metrics
Key metrics and results.

## What Worked
What went well.

## What Didn't Work
What could be improved.

## Learnings
Key insights from this sprint.

## Next Sprint
Focus for next iteration.
```
