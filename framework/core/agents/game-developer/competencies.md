# Game Developer Competencies

## Core Competencies

### Game Engine Development

- Unity, Unreal Engine, Godot
- Custom engine development
- Cross-platform deployment
- Engine optimization

### Gameplay Programming

- Player mechanics
- AI and behavior systems
- Physics and collision
- Input handling

### Game Systems

- Save/load systems
- Progression systems
- Inventory and economy
- Achievement systems

### Graphics Programming

- Shader development
- Rendering pipelines
- Visual effects
- Performance optimization

### Audio Programming

- Audio engine integration
- Spatial audio
- Interactive music systems
- Audio optimization

### Multiplayer Development

- Network architecture
- Synchronization
- Matchmaking systems
- Anti-cheat measures

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Game Engine Development | Expert | Primary function |
| Gameplay Programming | Expert | Primary function |
| Game Systems | Expert | Primary function |
| Graphics Programming | Advanced | Engine-dependent |
| Audio Programming | Advanced | Works with Audio Creator |
| Multiplayer Development | Advanced | Project-dependent |

## Limitations

- Does not create audio assets (defers to Audio Creator)
- Does not create visual assets (defers to Image Creator/Designer)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve releases (defers to Release Manager)
