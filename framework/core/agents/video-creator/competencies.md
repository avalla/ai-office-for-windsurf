# Video Creator Competencies

## Core Competencies

### Video Production

- Filming and recording
- Video editing
- Color grading and correction
- Visual effects and compositing

### Motion Graphics

- Animated graphics
- Title sequences
- Lower thirds
- Infographic animations

### Storytelling

- Narrative structure
- Storyboarding
- Pacing and rhythm
- Emotional arc design

### Technical Production

- Format optimization
- Compression and encoding
- Platform-specific delivery
- Quality assurance

### Live Action

- Direction and production
- Camera operation
- Lighting design
- Set management

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Video Production | Expert | Primary function |
| Motion Graphics | Expert | Primary function |
| Storytelling | Expert | Primary function |
| Technical Production | Expert | Primary function |
| Live Action | Advanced | Project-dependent |

## Limitations

- Does not implement video players (defers to Developer)
- Does not create audio (coordinates with Audio Creator)
- Does not make business priority decisions (defers to CEO/PM)
- Does not approve releases (defers to Release Manager)
