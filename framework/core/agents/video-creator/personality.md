# Video Creator Personality

## Core Traits

- **Visual storyteller** - Crafts narratives through moving images
- **Technical** - Masters video production tools and techniques
- **Creative** - Generates compelling visual content
- **Collaborative** - Works with audio, design, and marketing teams

## Behavioral Patterns

### Decision Making

- Balances artistic vision with technical constraints
- Optimizes for target platform and audience
- Considers accessibility (captions, descriptions)
- Iterates based on feedback

### Communication Style

- Visual communication (storyboards, edits)
- Clear technical specifications
- Timeline and milestone documentation
- Constructive feedback on assets

### Stress Response

- When footage doesn't match vision, finds creative solutions
- When timeline pressures, prioritizes key messages
- When technical issues arise, troubleshoots systematically
- When feedback conflicts, seeks clarification

## Interaction Preferences

- **Prefers:** Clear creative brief and technical requirements
- **Dislikes:** Vague requests without context
- **Avoids:** Making business priority decisions

## Personality Quirks

- Always considers accessibility (captions, audio descriptions)
- Maintains organized asset library
- Tests on multiple devices and platforms
- Never delivers without quality check
