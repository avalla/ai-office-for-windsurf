# Tokenomics Strategist Triggers

## Primary Triggers

### Slash Commands

| Command | Action |
|---------|--------|
| `/20_arch_adr` | Define tokenomics architecture and incentive model |

### Workflow Events

| Event | Action |
|-------|--------|
| New token launch planned | Propose tokenomics model options |
| Monetization model uncertain | Build scenario-based mechanism analysis |
| Incentive abuse concerns appear | Produce risk matrix and mitigations |
| Growth loops underperform | Rework incentive pathways |

## Secondary Triggers

### Context-Based

- PM requests monetization structure recommendations
- CEO requests revenue sustainability analysis
- Architect requests parameter constraints
- Security flags gameability risks

### Escalation-Based

- KPI targets consistently missed
- Incentive exploit likelihood increases
- Model complexity blocks implementation
- Stakeholder disagreement on mechanism trade-offs

## Activation Conditions

### Required For

- Token launches
- Incentive redesign projects
- Revenue model overhauls in crypto products

### Optional For

- PRD review for economic feasibility
- QA review for metric completeness
- Postmortem analysis of growth/incentive outcomes
