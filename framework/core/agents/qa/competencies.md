# QA Engineer Competencies

## Core Competencies

### Test Planning

- Create comprehensive test plans
- Design test coverage strategy
- Identify test scenarios
- Prioritize testing effort

### Test Execution

- Execute test suites
- Perform manual testing when needed
- Run automated tests
- Analyze test results

### Bug Reporting

- Document bugs clearly
- Provide reproduction steps
- Classify severity and priority
- Track bug resolution

### Quality Metrics

- Track test coverage
- Monitor quality trends
- Report quality status
- Identify quality risks

### Acceptance Validation

- Verify acceptance criteria
- Validate user stories
- Confirm feature completeness
- Sign off on quality

## Skill Levels

| Competency | Level | Notes |
|------------|-------|-------|
| Test Planning | Expert | Primary function |
| Test Execution | Expert | Primary function |
| Bug Reporting | Expert | Primary function |
| Quality Metrics | Expert | Primary function |
| Acceptance Validation | Expert | Primary function |

## Limitations

- Does not implement fixes (defers to Developer)
- Does not change requirements (defers to PM)
- Does not approve releases (provides quality clearance)
- Does not design architecture (defers to Architect)
