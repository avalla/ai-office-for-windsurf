# Provocation Director MCP Adapters

## Available Adapters

| Adapter | Server | Usage |
|---------|--------|-------|
| `sequential-thinking` | `sequential-thinking` | Multi-path concept exploration and trade-off analysis |
| `fetch` | `fetch` | Reference collection and category narrative audits |
| `stitch` | `stitch` | Rapid concept visualization and design alternatives |
| `runcomfy` | `runcomfy` | Experimental visual style and mood exploration |

## Adapter Usage Patterns

### sequential-thinking

When Used:
- Develop and compare disruptive concept branches
- Resolve novelty vs clarity vs safety trade-offs

### fetch

When Used:
- Research campaign references and narrative patterns
- Check differentiation against current market conventions

### stitch

When Used:
- Generate and compare concept-aligned UI/visual layouts
- Explore alternative visual directions quickly

### runcomfy

When Used:
- Generate key style frames and bold visual references
- Prototype mood and artistic direction before production

## Adapters NOT Used

Provocation Director does NOT use:
- `snyk` - Security scanning belongs to Security Specialist
- `supabase` - Database operations belong to Developer/Architect
- `revenuecat` - Revenue operations not in scope

## Adapter Constraints

- No ownership of code implementation
- No direct production deployment authority
- Focused on concept direction and creative coherence
